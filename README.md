# Bahram's dotfiles

## Vim

```bash
mv ~/.vim/ ~/.oldvim  # Back up current configuration if exists!

git clone https://github.com/greatbahram/dotfiles.git

cp -r  dotfiles/.vim ~/
```

*Enjoy it*!

## Git

```shell
cd dotfiles
./setup.sh
```
